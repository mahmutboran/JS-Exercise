Exercise-7: 

    write a code that returns js fibonacci numbers

    output :

    fibonacci(7)-->21

Solute:

                    function fibonacci(n) {
                        if (n===1){return 1;}
                        else if (n===2) {return 1}
                        else { return fibonacci(n-2)+fibonacci(n-1);
                         }
                    }
                    console.log(fibonacci(7));
